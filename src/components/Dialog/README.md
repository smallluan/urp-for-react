### Dialog Props
|参数名|是否必填|参数类型|默认值|说明|
|--|--|--|--|--|
|**className**||||透传类名|
|**style**||||透传样式|
|**width**||||宽度|
|**theme**||||主题|
|**title**||||标题内容|
|**cancelBtn**||||取消按钮（类型为字符串修改按钮文字，布尔值修改显隐性，对象类型透传值 button）|
|**confirmBtn**||||确认按钮（类型为字符串修改按钮文字，布尔值修改显隐性，对象类型透传值 button）|
|**closeBtn**||||右上角的关闭按钮（类型为字符串修改按钮文字，布尔值修改显隐性，对象类型透传值 button）|
|**visible**||||是否显示（受控）(overlay属性)|
|**children**||||内容|
|**content**||||内容，同上，优先级小于 children|
|**destoryOnClose**||||关闭时销毁元素|
|**zIndex**||||层级高度(overlay属性)|
|**attachBody**||||是否挂载到body(overlay属性)|
|**onCloseBtnClick**||||右上角关闭点击事件|
|**onConfirm**||||确认按钮点击事件|
|**onCancel**||||取消按钮点击事件|
|**onOverlayClick**||||遮罩层点击事件|
|**onEscKeydown**||||按下 Esc 健事件|
