|参数名|是否必填|参数类型|默认值|说明|
|--|--|--|--|--|
|**className**||||图片类名|
|**style**||||图片样式|
|**alt**||||图片描述|
|**objectFit**||||图片填充模式|
|**src**||||图片链接|
|**fallback**||||图片加载失败后显示的图片地址（可以防止重复触发 error）|
|**lazy**||||是否开启懒加载|
|**shape**||||图片形状|
|**loading**||||图片加载时内容|
|**error**||||图片加载失败显示的内容|
|**overlayContent**||||图片浮层内容|
|**overlayTrigger**||||何时触发浮层|
|**onError**||||图片加载失败回调|
|**onLoad**||||图片加载完成回调|
