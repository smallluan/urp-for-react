|参数名|是否必填|参数类型|默认值|说明|
|--|--|--|--|--|
|**className**||||透传类名|
|**style**||||透传样式|
|**value**||||当前的评分（受控属性）|
|**defaultvalue**||||当前的评分（非受控属性）|
|**count**||||评分数量|
|**disabled**||||是否禁用|
|**allowPartial**||||是否允许部分选择（精度 0.1）|
|**clearable**||||是否允许再次点击当前评分清空|
|**color**||||图标颜色|
|**texts**||||不同分数的辅助文字|
|**gap**||||图标间距|
|**size**||||尺寸|
|**onChange**||||评分变化时触发|
