## Swiper Props
|参数名|是否必填|参数类型|默认值|说明|
|--|--|--|--|--|
|**className**||||类名|
|**style**||||样式|
|**autoplay**||||是否自动播放|
|**current**||||当前展示的轮播图（受控属性）|
|**defaultCurrent/**||||当前展示的轮播图（非受控属性）|
|**direction**||||轮播图滑动方向，有横向和纵向
|**duration**|||||滑动动画时长|
|**interval**||||轮播间隔|
|**loop**||||是否循环播放|
|**stopOnHover**||||悬浮时是否暂停播放|
|**filterChildNode**||||是否过滤有效的item子节点(USwiper.Item)|
|**onChange**||||轮播切换时触发|
