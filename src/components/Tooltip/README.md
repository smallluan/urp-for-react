## Props（选择性继承 popup 属性）
|参数名|是否必填|参数类型|默认值|说明|
|--|--|--|--|--|
|**className**||||透传类名|
|**style**||||透传样式|
|**position**||||位置（popup属性）|
|**arrow**||||是否显示箭头（popup属性）|
|**trigger**||||如何触发（popup属性）|
|**content**||||内容文字（popup属性）|
|**theme**||||主题|
|**popupProps**||||popup属性透传|
|**children**||||被挂载的元素|
